"""
Tool Training Examples for Enigma AI
=====================================

This file contains training examples that teach the AI how to use tools.
The AI should learn to:
  1. Recognize when a tool should be used
  2. Output the correct tool call format
  3. Understand and respond to tool results
  4. Use tools naturally in conversation

Format:
  Q: User query
  A: AI response with tool calls and results

Tool Call Format (STANDARDIZED JSON):
  <tool_call>{"tool": "tool_name", "params": {"param1": "value1", "param2": "value2"}}</tool_call>
  <tool_result>{"tool": "tool_name", "success": true, "result": "Tool execution result text"}</tool_result>

Special Tokens:
  <tool_call> - Start of JSON tool invocation
  </tool_call> - End of tool invocation
  <tool_result> - Start of JSON tool result
  </tool_result> - End of tool result

Note: This format is consistent with tool_executor.py parsing and tool_definitions.py schemas.
"""
