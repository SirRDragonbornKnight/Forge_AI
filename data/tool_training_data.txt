Here's a Python function to calculate Fibonacci numbers:

```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```

This uses a recursive approach. You can call it like `fibonacci(10)` to get the 10th Fibonacci number.
