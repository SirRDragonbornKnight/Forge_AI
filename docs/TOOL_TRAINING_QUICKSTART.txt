STEP 1: UNDERSTAND THE FORMAT (2 minutes)
================================================================================

All training examples use this format:

    Q: User's question
    A: AI acknowledgment
    <tool_call>{"tool": "tool_name", "params": {"key": "value"}}</tool_call>
    <tool_result>{"tool": "tool_name", "success": true, "result": "..."}</tool_result>
    AI's response with the result


================================================================================
STEP 2: CREATE TRAINING EXAMPLES (10-30 minutes)
================================================================================

Open: data/tool_training_data.txt

Add examples for each tool you want the AI to use.
Minimum: 10 examples per tool (50+ recommended)


TEMPLATE FOR EACH TOOL:
-----------------------

# [TOOL NAME]

Q: [Request phrasing 1]
A: [Short acknowledgment]
<tool_call>{"tool": "[name]", "params": {[parameters]}}</tool_call>
<tool_result>{"tool": "[name]", "success": true, "result": "[result]"}</tool_result>
[Natural response]

Q: [Request phrasing 2 - different words, same intent]
A: [Different acknowledgment]
<tool_call>{"tool": "[name]", "params": {[different params]}}</tool_call>
<tool_result>{"tool": "[name]", "success": true, "result": "[result]"}</tool_result>
[Different natural response]

Q: [Request that fails]
A: [Acknowledgment]
<tool_call>{"tool": "[name]", "params": {[params]}}</tool_call>
<tool_result>{"tool": "[name]", "success": false, "error": "[error]"}</tool_result>
[Graceful error response]

================================================================================
STEP 4: TRAIN THE MODEL (5-30 minutes)
================================================================================

Run training:

    python run.py --train

Training will:
  1. Load all data from data/ folder
  2. Train on tool examples
  3. Save the model to models/


================================================================================
STEP 5: TEST (5 minutes)
================================================================================

Run the AI:

    python run.py --run

Test with tool requests:
  > Search for Python tutorials
  > Generate an image of a cat
  > What files are in the current folder?
  > Remind me to take a break in 30 minutes

The AI should respond with proper tool calls!


================================================================================
TROUBLESHOOTING
================================================================================

AI doesn't call tools:
  - Add more training examples (50+ per tool)
  - Make sure format is exactly right
  - Check for typos in tool names

AI calls wrong tool:
  - Add more variety in phrasings
  - Make sure different tools have distinct trigger words

Tool call has wrong format:
  - Check JSON syntax in training data
  - Ensure params match expected parameters

Tools don't execute:
  - Check if tool module is enabled
  - Verify tool is registered in tool_registry.py
